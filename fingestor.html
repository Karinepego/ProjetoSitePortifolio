Projeto - Fingestor

Este é um projeto feito por encomenda pra um cliente se trata de um software desktop feito com python e tkinter.

O Fingestor é uma aplicação desktop desenvolvida para facilitar a gestão financeira pessoal. Com uma interface intuitiva construída usando Python e Tkinter, o Fingestor permite que os usuários registrem suas receitas e despesas de forma simples e eficiente.

possui algumas funcionalidades e ainda esta em fase de desenvolvimento.



Desenvolvido com:


Backend & Microsserviços:
Go (Golang), Node.js / NestJS, Python, TypeScript.

Frontend:
React.js,Tailwind CSS, Vite, Axios.

Infraestrutura & Dados:
Docker, RabbitMQ, MongoDB.

Arquitetura de microsserviços:
Generative AI: Integração com LLMs (Google Gemini API).
JWT & Bcrypt: Autenticação e Segurança.
Git/GitHub: Controle de versão e fluxo de Pull Requests.
Docker: Orquestração de containers.
